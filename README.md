# Symbols examples
Example [Symbols](https://symbols.dev/) components.

Open in [Codesandbox](https://go.symbo.ls/example-codesandbox) | [VS Code](https://go.symbo.ls/example-vscode)

### Setup

1. Clone the repo
```
git clone git@github.com:symbo-ls/example.git
```

2. Install scripts
```
yarn
```

3. Run the project
```
yarn start
```
